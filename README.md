# TelecomX-Analysis2

Este repositorio contiene el análisis predictivo de evasión de clientes (churn) para TelecomX, utilizando los datos tratados del desafío anterior (`TelecomX-Churn-Analysis`). El proyecto incluye preparación adicional de datos, modelado de machine learning, análisis de importancia de variables y conclusiones estratégicas para reducir el churn.

## Estructura
- `/data`: Archivos de datos, incluyendo `TelecomX_Data_with_daily.csv`.
- `/img`: Imágenes de gráficos para documentación.
- `/notebooks`: Notebooks con los pasos del análisis predictivo.
